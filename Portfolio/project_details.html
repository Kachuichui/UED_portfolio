Study 1

Enhancing the Mobile Banking Experience

Project Details

Role
Lead UX/UI Designer
Duration
3 Months
Tools
Figma, Miro, UsabilityHub

Overview

This project focused on redesigning the user interface and improving the user experience of a mobile banking application. The goal was to create a more intuitive, efficient, and visually appealing app that would increase user engagement and satisfaction, drawing inspiration from Tang Dynasty aesthetics for a unique feel.

Problem Statement

The existing mobile banking app had several usability issues, including a cluttered interface, confusing navigation, and a lack of visual hierarchy. Users reported difficulty in performing common tasks such as transferring funds, paying bills, and viewing transaction history.

Process

Our design process involved several key stages to ensure a user-centric solution.

  1. 壹.User Research: Conducted interviews and testing to identify pain points and understand user needs.
  2. 貳.Information Architecture: Restructured the app's IA for a more logical and intuitive flow.
  3. 叁.Wireframing & Prototyping: Developed low and high-fidelity wireframes, leading to an interactive prototype.
  4. 肆.Visual Design: Created a clean, modern design system aligned with the brand's identity, infused with Tang elements.
  5. 伍.Usability Testing: Validated the new design through rigorous testing and iterated based on feedback.

Solution

The redesigned app features a streamlined interface with clear visual cues and intuitive navigation. Key improvements include a simplified dashboard, improved navigation, clear task flows, and an enhanced visual design inspired by Tang Dynasty art.

Outcome

The redesign yielded significant positive results, validating our user-centered approach.
